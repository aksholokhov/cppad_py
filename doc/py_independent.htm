Declare Independent Variables and Start Recording

Syntax
ax = cppad_py.independent(x)

x
This argument is a numpy vector with float elements. It specifies the number of independent variables and their values during the recording. We use n = x.size to denote the number of independent variables.

ax
The result is a numpy vector with a_double elements. This is the vector of independent variables. It has size n and for i = 0 to n-1
     ax[i].value() == x[i]
Purpose
This starts a recording of the a_double operations. This recording is terminated, and the information is stored, by calling the a_fun constructor . It is terminated, and the information is lost, by calling abort_recording .

Example
All of the python a_fun examples use this function.
