Syntax
yf = runge4.one_step(f, ti, yi, h)

f
This is a function that evaluates the ordinary differential equation using the syntax yp = f( t , y ) where t # is the current time, y is the current value of @(@ y(t) @)@, and yp is the current derivative @(@ y^{(1)} (t) @)@.

ti
This is the initial time for the Runge-Kutta step.

yi
This is the value of @(@ y(t) @)@ at the initial time.

h
This is the step size in time; i.e., the time at the end of the step minus the initial time.

yf
This is the approximate solution for @(@ y(t) @)@ at the final time. This solution is 4-th order accurate in time @(@ t @)@; e.g., if @(@ y(t) @)@ is a polynomial in @(@ t @)@ of order four or lower, the solution has no truncation error, only round off error.

def one_step(f, ti, yi, h) :
     k1     = h * f(ti,           yi)
     k2     = h * f(ti + h / 2.0, yi + k1 / 2.0)
     k3     = h * f(ti + h / 2.0, yi + k2 / 2.0)
     k4     = h * f(ti + h,       yi + k3 )
     yf     = yi + (k1 + 2.0 * k2 + 2.0 * k3 + k4) / 6.0
     return yf
